# Typing Speed Test
Try it out [here](http://typingspeedtest.surge.sh/).

## Setup
The usual: 
- `git clone`
- `cd typingspeedtest`
- `npm start`

## Technologies
- Vanilla JavaScript
- HTML / CSS

## App features
- Three difficulty settings
- Error count
- Final word count
- Final score (wpm)
- Score to beat (previous score)
- Colour-coded typing test area border
- Stopwatch
- "Start over" button
